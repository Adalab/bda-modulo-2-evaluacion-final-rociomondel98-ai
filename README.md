· Contenido del repositorio 

Este repositorio contiene dos ficheros .sql que constan de la resolución del ejercicio. El archivo codigo_bonus incluye además el código del ejercicio bonus.
También incluye el enunciado del ejercicio en formato PDF.

· Detalles del código

En los archivos de código además hay una breve explicación de las decisiones tomadas para resolver estos ejercicios así como los enunciados. 

El código incluye el uso de joins, consultas básicas así como avanzadas, operadores de filtro y subconsultas. 

El ejercicio utiliza la base de datos Sakila.
